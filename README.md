# **Aplicativo de mensagens para a empresa SkyBlue.**

##### Buscando atender a demanda da emprea *SKYBLUE*, desenvolvi uma parte de um aplicativo voltado para enviar mensagens, foi feito usando a base do programa *ECLIPSE*, primeiro foi usado uma importação para pedri aos clientes que digitem seu nome, e assim que eles digitam seu nomes, o aplicativo envia uma mensagem desejando boas vindas ao sistema *SKYBLUE*. Entre os comandos usados estão : 
1. import java.util.Scanner; Essa classe permite ler dados digitados pelo usuário.
2. Scanner sc = new Scanner(System.in); epresenta a entrada padrão (teclado).
3. System.out.println("Digite seu nome: "); Mostra uma mensagem na tela pedindo o nome do usuário.
4. String nome = sc.next(); lê uma palavra (até o primeiro espaço)
5. System.out.println("Seja bem vindo a SkyBlue "+nome+" !!!"); "Seja bem vindo..." → texto fixo, + nome → concatena (junta) com o nome digitado.


# **Principais IDE's e suas diferenças

## 1. Visual Studio Code (VS Code)
É um editor de código, mas com extensões vira uma IDE completa.

* Diferencial: Extremamente leve e versátil. Você instala apenas o que precisa. Serve para quase tudo (Web, Python, JS, C#).
* Ponto fraco: Se entupir de extensões, ele pode ficar lento, e a configuração inicial é mais manual. 

## 2. IntelliJ IDEA
Focada principalmente em Java e ecossistema JVM (Kotlin, Scala).

* Diferencial: É considerada a IDE mais "inteligente". O autocomplete e as sugestões de refatoração de código são quase mágicos.
* Ponto fraco: Consome muita memória RAM e a versão completa (Ultimate) é paga.

## 3. Visual Studio (não confunda com o VS Code)
A IDE parruda da Microsoft para o ecossistema .NET e C++.

* Diferencial: É uma ferramenta completa "out-of-the-box". Já vem com tudo para criar softwares complexos para Windows, jogos em Unity e aplicações Enterprise.
* Ponto fraco: É muito pesada (ocupa dezenas de GB no disco) e roda apenas em Windows (a versão Mac foi descontinuada).

## 4. PyCharm
Específica para Python (da mesma empresa do IntelliJ).

* Diferencial: Tem as melhores ferramentas do mercado para Data Science e Django. Ela entende o contexto do código Python melhor que qualquer outra.
* Ponto fraco: Assim como o IntelliJ, é pesada e pode ser exagerada para scripts simples.

## 5. Eclipse
Uma das mais antigas e tradicionais, muito usada no meio acadêmico e em sistemas legados.

* Diferencial: É totalmente gratuita e extremamente customizável através de plugins acumulados por décadas.
* Ponto fraco: A interface parece datada e é conhecida por ser um pouco instável ou confusa para iniciantes hoje em dia.
